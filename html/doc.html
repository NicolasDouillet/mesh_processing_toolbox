Example #9 : convex hull

nb_vtx = 128;
X = 2*(rand(nb_vtx,1)-0.5);
Y = 2*(rand(nb_vtx,1)-0.5);
Z = 2*(rand(nb_vtx,1)-0.5);

Rho = X.^2 + Y.^2 + Z.^2;
i = Rho <= 1;
X = X(i);
Y = Y(i);
Z = Z(i);
V = cat(2,X,Y,Z);

plot_point_set(V,'o','y',4);
axis equal;
view(3);

[V,Qh] = quick_hull(V);
plot_mesh(V,Qh);
axis equal;
view(3);

Mesh quick hull computed in 6.053576e-01s.
